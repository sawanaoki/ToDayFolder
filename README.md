# ToDayFolder

デスクトップの整理整頓を自動化するための一時フォルダ作成＆自動ファイル移動ツールです。
実行ファイルやショートカットを利用して、今日の日付のフォルダを瞬時に生成し、古い作業フォルダは自動で圧縮・削除してくれます。NAS環境などでの一元管理にも対応しています。

## 概要

- **ダブルクリックだけ**で「今日の日付（YYYY-MM-DD）」のフォルダを作成し、その場で開きます。
- **ファイルをドラッグ＆ドロップ**するだけで、今日のフォルダを作成した上で内部へアイテムを自動整頓（移動）します。
- 不要になった数日前の古いフォルダを**自動でZIP圧縮**したり、**完全削除**するメンテナンス機能も備わっています。

## 使い方

EXEファイル（`ToDayFolder.exe`）の配置場所やショートカットの作成方法によって、用途に合わせて柔軟に運用できます。

### 1. 基本的な使い方
- **ダブルクリックする**: `ToDayFolder.exe` 本体をダブルクリックすると、同じ階層に「今日の日付」のフォルダを作成して自動で開きます。同時に、設定された日数を過ぎた古いフォルダのお掃除（圧縮・削除）が裏で実行されます。
- **ドラッグ＆ドロップする**: `ToDayFolder.exe` のアイコンに向かってファイルやフォルダを直接ドラッグ＆ドロップすると、今日のフォルダを作成し、その中にドロップしたアイテムを移動します。

※ NASなどの共有ドライブにEXEファイルを置き、複数人でショートカットを共有して利用することも可能です（同じタイミングでファイルを入れた場合は連番が振られ、上書きはされません）。

### 2. 設定の変更
- **本体を Shift + ダブルクリック**: `ToDayFolder.exe` を Shift キーを押しながら起動すると設定画面が開きます。
  - **ドロップ後にフォルダを開く**: アイテム移動完了後、自動でエクスプローラーを開きます。
  - **ドライブが違う場合でも、コピーせず移動**: 通常、別ドライブ間のドラッグ＆ドロップは「コピー」になりますが、強引に「移動（切り取りペースト）」扱いにします。
  - **今日のフォルダの作成先をTodayFolders以下にする**: 日付フォルダが散らからないよう、`TodayFolders` という親フォルダを一つ作成し、その中に日付フォルダ群を作成します。
  - **過去のTodayFolderを圧縮する (○日後)**: 指定日数が経過した過去のフォルダをZIPに圧縮し、元のフォルダを削除します。(0で即時)
  - **過去のTodayFolderを削除する (○日後)**: 指定日数が経過した過去のフォルダ（および圧縮されたZIPファイル）を完全削除します。(0で即時)

> **[注意]** NASなどで運用の場合、ゴミ箱機能を経由せず手裏剣完全削除となるため日数の設定にはご注意ください。

### 3. ショートカットの活用と注意点
デスクトップにショートカットを作成して利用するのが便利ですが、ショートカット経由でShiftキーを押しても設定画面が開いてしまう誤動作を防ぐため、以下の設定を推奨します。

1. 作成したショートカットを右クリックし、「プロパティ」を開く。
2. 「リンク先」の末尾に、半角スペースを空けてから `--nosettings` と追記する。
   - `例: "D:\ソフト開発\ToDays\dist\ToDayFolder.exe" --nosettings`
3. このショートカットを使えば、誤ってShiftキーを押したまま起動（またはドロップ）しても、設定画面が開くことなく通常のフォルダ作成やファイル移動がサクサク行えます。

## ビルド方法

Python環境（Windows）で開発を行っています。
Pillow等の一部のライブラリを利用してアイコンを作成しています。

```bash
# 依存ライブラリのインストール
python -m pip install Pillow pyinstaller

# 実行可能なexeへのビルド
pyinstaller --noconsole --onefile --icon=folder.ico --name ToDayFolder --clean TodayFolderCreator.py
```
